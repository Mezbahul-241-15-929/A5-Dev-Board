Question-1: What are the different ways to select an element in the DOM?

  • document.getElementById("id")
  • document.getElementsByClassName("class")
  • document.getElementsByTagName("tag")
  • document.querySelector("selector")
  • document.querySelectorAll("selector")

    • Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML: Gets or sets HTML content, including tags.
  • innerText: Gets or sets visible text, ignoring hidden elements.
  • textContent: Gets or sets all text, including hidden elements, but without HTML.

    • Question-3: What is event delegation in the DOM?

    A technique where a parent element handles events for its child elements using event bubbling.

    Question-4: What is event bubbling in the DOM?

    When an event on a child element propagates up to its parent elements.

    Question-5: How do you create, add, and remove elements using JavaScript?

  • Create: let a = document.createElement("div");
  • Add: parent.appendChild(a); or parent.append(a);
  • Remove: a.remove(); or parent.removeChild(a);
